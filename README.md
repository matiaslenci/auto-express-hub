# AgenciaExpress

Plataforma de gestión para agencias de vehículos.

## Información del Proyecto

Esta aplicación permite a las agencias de vehículos gestionar su inventario, visualizar estadísticas y conectar con clientes a través de WhatsApp.

## Tecnologías utilizadas

- Vite
- TypeScript
- React
- shadcn/ui
- Tailwind CSS

## Configuración Local

Para ejecutar el proyecto localmente:

1. Clonar el repositorio
2. Instalar dependencias:
   ```bash
   npm install
   ```
3. Iniciar el servidor de desarrollo:
   ```bash
   npm run dev
   ```

## Construcción

Para generar la versión de producción:
```bash
npm run build
```
